# Moodle Upload and QA Checklist

## Before Exporting from Lumi

- Activity title is clear and matches the Moodle module.
- Instructions tell students exactly what to do.
- Every question has useful feedback.
- Images have alt text.
- Video captions or transcript are available if using video.
- Activity fits the intended time limit.
- Scoring is enabled only when needed.

## Export from Lumi

1. Open the activity in Lumi.
2. Choose `File > Export`.
3. Select H5P package format.
4. Save the `.h5p` file with a stable name:
   - `module-01-knowledge-check.h5p`
   - `module-01-branching-scenario.h5p`
   - `module-01-interactive-presentation.h5p`

## Upload to Moodle

1. Open the Moodle course.
2. Turn editing on.
3. Add an activity or resource.
4. Choose H5P activity, or upload to the Content bank first.
5. Upload the `.h5p` package.
6. Set grade, attempts, and completion rules.
7. Save and preview as a student.

## Student Preview Test

- Activity opens without missing libraries.
- Instructions are visible before the first interaction.
- Buttons and inputs work on desktop and mobile widths.
- Feedback appears after incorrect and correct answers.
- Score reports to Moodle if grading is enabled.
- Completion triggers correctly.
- Any embedded media loads and plays.
