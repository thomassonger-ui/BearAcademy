# Lumi H5P Activity Kit for Moodle

Use this kit to plan, build, test, and upload H5P activities made in Lumi for a Moodle course.

## Recommended Workflow

1. Pick the activity type from `activity-plan.md`.
2. Fill in the matching template in `templates/`.
3. Build the activity in Lumi.
4. Export as `.h5p`.
5. Upload to Moodle as an H5P activity or into the content bank.
6. Test using the checklist in `moodle-upload-checklist.md`.

## Best Activity Mix

For one Moodle module, a strong default mix is:

- 1 interactive presentation for guided instruction
- 1 knowledge check quiz
- 1 drag-and-drop or image hotspot activity for visual practice
- 1 branching scenario for application
- 1 summary or reflection activity for consolidation

Keep most activities under 5 minutes. H5P works best when each activity has one clear job.

## Files

- `activity-plan.md`: Suggested activity set and learning purpose.
- `templates/`: Copy-and-fill templates for each H5P type.
- `examples/`: Small sample activities showing the expected level of detail.
- `moodle-upload-checklist.md`: QA checklist before releasing to students.
