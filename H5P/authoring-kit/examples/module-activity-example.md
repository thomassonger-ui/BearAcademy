## Questions

### Question 1

- Type: Multiple choice
- Prompt: Which option is the clearest opening sentence for a workplace request?
- Correct answer: Please send the revised schedule by Friday at 2:00 p.m.
- Distractors:
  - I need that thing soon.
  - Can you get it to me whenever?
  - The schedule is probably something we should revisit.
- Feedback if correct: Correct. The request names the task, deadline, and expected action.
- Feedback if incorrect: Look for the option that gives a specific task and deadline.

### Question 2

- Type: True-false
- Prompt: A clear message should usually include the action you want the reader to take.
- Correct answer: True
- Feedback if correct: Correct. The reader should know what to do next.
- Feedback if incorrect: A message may be polite and accurate but still unclear if the next action is missing.

### Question 3

- Type: Multiple choice
- Prompt: Which detail would most improve this message? "Can you review the file?"
- Correct answer: The file name and due date.
- Distractors:
  - A longer greeting.
  - A more casual tone.
  - A quote about teamwork.
- Feedback if correct: Correct. The reader needs to know which file and when the review is needed.
- Feedback if incorrect: Think about what information the reader needs to complete the task.
